# CS2_Refiller
CS2 Refiller - плагин для CS2, автоматически восполняющий здоровье, броню и патроны игроку за убийство или помощь в убийстве.

## Описание плагина: CS2 Refiller - плагин для CS2, автоматически восполняющий здоровье, броню и патроны игроку за убийство или помощь в убийстве.

### Конфигурационные параметры:

AssistRefill (true) - Награждать ассистента таким же вознаграждением как убийцу

HealthRefill ("25") - Восстановление здоровья: "all" до 100, "0" выкл, число для добавления HP

AmmoRefill ("current") - Восстановление патронов: "all" все оружия, "current" только активное, "off" выкл

ArmorRefill ("15") - Восстановление брони: "all" до 100, "0" выкл, число для добавления

### Консольные команды:

css_refiller_help - Показать справку

css_refiller_settings - Показать настройки

css_plugins reload CS2Refiller - Перезагрузить конфигурацию
