# CS2_Refiller
Плагин автоматически восстанавливает здоровье, патроны и броню игроку, совершившему убийство (и, опционально, ассистенту). Также имеется функция автоматического пополнения резервных патронов, когда игрок пытается перезарядиться или стрелять, но резерв пуст. Это позволяет избежать ситуаций, когда игрок остаётся без боеприпасов в разгар боя.

# Требования
```
CounterStrikeSharp API версии 362 или выше
.NET 8.0 Runtime
```

# Конфигурационные параметры
```
css_refiller_assist <0/1>, def.=1 – Награждать ли ассистента за помощь в убийстве: 1 – ассистент получает такое же восстановление, как и убийца; 0 – только убийца.
css_refiller_health <"all"/"0"/число>, def.="25" – Восстановление здоровья: "all" – установить здоровье на 100 HP; "0" – не восстанавливать; число (например, "20") – добавить указанное количество HP (но не более 100).
css_refiller_ammo <"all"/"current"/"off">, def.="current" – Восстановление боеприпасов: "all" – восстановить патроны во всех оружиях в инвентаре; "current" – только в активном оружии; "off" – не восстанавливать.
css_refiller_armor <"all"/"0"/число>, def.="15" – Восстановление брони: "all" – установить броню на 100; "0" – не восстанавливать; число (например, "10") – добавить указанное количество брони (но не более 100).
css_refiller_autorefillclip <0/1>, def.=1 – Автоматически пополнять резервные патроны: 1 – включено, 0 – выключено. При включении работает следующим образом:
При нажатии кнопки перезарядки (R), если резерв пуст, он пополняется до максимума. При попытке стрельбы (нажатие атаки), если магазин пуст и резерв пуст, резерв также пополняется. После этого игра автоматически начинает перезарядку.
css_refiller_loglevel <0-5>, def.=4 – Уровень логирования: 0-Trace, 1-Debug, 2-Information, 3-Warning, 4-Error, 5-Critical.
```

# Консольные команды
```
css_refiller_help – Показать подробную справку по плагину с описанием всех команд и примеров.
css_refiller_settings – Показать текущие настройки плагина, включая значения всех параметров и количество активных игроков.
css_refiller_test – Вывести в чат текущие настройки и инструкцию по проверке работы плагина (для игрока).
css_refiller_reload – Перезагрузить конфигурацию из файла и применить новые настройки.
css_refiller_setassist <0/1> – Включить (1) или выключить (0) награду ассистенту (параметр css_refiller_assist).
css_refiller_sethealth <all/0/число> – Установить значение параметра css_refiller_health. Допустимые значения: all, 0, положительное число.
css_refiller_setammo <all/current/off> – Установить значение параметра css_refiller_ammo. Допустимые значения: all, current, off.
css_refiller_setarmor <all/0/число> – Установить значение параметра css_refiller_armor. Допустимые значения: all, 0, положительное число.
css_refiller_setautorefillclip <0/1> – Включить (1) или выключить (0) автоматическое пополнение резервных патронов (параметр css_refiller_autorefillclip).
css_refiller_setloglevel <0-5> – Установить уровень логирования (параметр css_refiller_loglevel).
```
